# 设备缺陷统计管理系统 V2.0

## 项目简介
本项目是一个用于自动化统计和管理设备缺陷记录的桌面应用程序。它能够自动从 Word 文档中提取缺陷处理记录，汇总到 Excel 表格中，并提供直观的统计分析仪表盘。

## 主要功能

### 1. 自动化数据采集
- **批量读取**：自动扫描指定目录下的 Word 文档（.doc/.docx）。
- **智能解析**：提取文档中的表格数据，自动识别有效内容。
- **增量同步**：
  - 自动识别新增的 Word 文档并追加数据。
  - **智能清理**：自动检测被删除的 Word 文档，并从 Excel 汇总表中移除对应数据。

### 2. 数据管理
- **Excel 汇总**：将分散的 Word 记录统一汇总到 Excel 表格中。
- **格式保持**：在写入 Excel 时自动保持模板格式（字体、边框、对齐方式等）。
- **序号管理**：自动维护数据序号，确保连续性。

### 3. 统计分析
- **可视化仪表盘**：
  - 缺陷总数、未销号、已销号实时统计。
  - 销号完成率饼图。
  - 缺陷类型分布柱状图。
- **交互式筛选**：支持按年份、月份筛选统计数据。
- **明细列表**：内置数据查看器，支持快速浏览和状态查看。

## 技术栈
- **编程语言**: Python 3.x
- **GUI 框架**: Tkinter + ttkbootstrap (Cosmo 主题)
- **数据处理**: pandas, openpyxl
- **Office 自动化**: pywin32 (win32com)
- **图表绘制**: matplotlib

## 环境依赖
请确保安装以下 Python 库：
```bash
pip install pandas openpyxl pywin32 matplotlib ttkbootstrap
```

## 使用说明
1. 确保已安装 Microsoft Word（程序依赖 Word COM 接口读取 .doc 文件）。
2. 运行 `启动程序.bat` 或直接执行 `run.py` 启动应用。
3. 在“数据采集”页面选择 Word 文档所在文件夹。
4. 点击“开始处理”或“同步并刷新”进行数据提取。
5. 切换到“统计分析”页面查看可视化报表。

## 注意事项
- 程序依赖本地安装的 Office Word 进行文档解析。
- 请勿在程序运行时打开目标 Excel 文件，以免写入失败。
